# 大众点评多模态检测数据集 - 样例结构说明
## 1. 数据规模
- 正样本店铺：50家（实战单位提供）
- 负样本店铺：49家（大众点评随机抽取）
- 总计：99家店铺的全模态信息

## 2. 7类数据模态（支持 jpg/png 双后缀）
| 模态编码 | 模态名称 | 支持格式 | 示例文件 |
|----------|----------|----------|----------|
| 1 | 封面配图 | jpg/png | 封面配图.jpg、封面配图.png |
| 2 | 全页配图 | jpg/png | 全页配图（1）.jpg、全页配图（2）.png |
| 3 | 详情页配图 | jpg/png | 详情页配图（1）.jpg |
| 4 | 商户自发布的动态配图 | jpg/png | 商户自发布的动态配图.png |
| 5 | 商户自发布的动态视频 | mp4 | 商户自发布的动态视频.mp4 |
| 6 | 评论区图片 | jpg/png | 评论区图片（1）.jpg |
| 7 | 评论区评论 | xlsx | XX店铺.xlsx（sheet名：评论区） |

## 3. 数据层级结构
### 负样本49家
不涉黄49家 → 第一批 → 人名文件夹（如程馨源（16家）） → 店铺文件夹
### 正样本50家
涉黄50家 → 50家店 → 人名文件夹（如程馨源（14家）） → 店铺文件夹

## 4. 数据说明
本数据集所有内容均为公开开源信息，仅包含店铺公开的配图、视频、评论等内容，无隐私信息，无需脱敏处理。
