# DianPing_PornDetect_MM
A Multimodal Dataset for Detection of DianPing Stores（大众点评店铺多模态检测数据集）

## 📜 数据集核心信息
### 1. 数据规模
- 正样本店铺：50家（实战单位提供）
- 负样本店铺：49家（大众点评随机抽取）
- 总计：99家店铺的全模态信息

### 2. 7类数据模态
| 模态编码 | 模态名称 | 格式 | 说明 |
|----------|----------|------|------|
| 1 | 封面配图 | jpg/png | 店铺主页封面图片 |
| 2 | 全页配图 | jpg/png | 店铺全页展示图片 |
| 3 | 详情页配图 | jpg/png | 店铺详情页图片 |
| 4 | 商户自发布的动态配图 | jpg/png | 店铺主动发布的动态图片 |
| 5 | 商户自发布的动态视频 | mp4 | 店铺主动发布的动态视频 |
| 6 | 评论区图片 | jpg/png | 用户评论区上传的图片 |
| 7 | 评论区评论 | xlsx | 用户评论内容（sheet名：评论区） |
